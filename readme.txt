Quote Tag plugin turns &lt;q&gt; tags into quotations to encourage semantically correct html. Works in all browsers.
 
== Description ==

Quote Tag plugin displays short quotations within &lt;q&gt; tags as quotes. For example, &lt;q&gt;Believe you can and you're
halfway there&lt;/q&gt; is displayed as "<em>Believe you can and you're halfway there</em>".

Quote Tag plugin works for pages, posts and custom posts using the tinymce's html editor.

== Installation ==

Install from wordpress plugins directory.

To install manually:

Upload unzipped plugin folder to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress
